EventStore PHP client
=====================

PHP client for [EventStore 3.x HTTP API](https://github.com/eventstore/eventstore/wiki/Getting-Started-HTTP)

Roadmap
-------

Development started in April 2014. Not ready for production use.

API can currently:

- Read from streams
- Navigate streams
- Read events
- Write to streams
- Delete streams
